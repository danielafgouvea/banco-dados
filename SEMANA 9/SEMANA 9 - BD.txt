Atividade Prática – SQL Avançado (INTERSECT e EXCEPT)

Disciplina: Modelagem de Banco de Dados
Semana: 9
Ferramenta: Supabase

Contexto (Situação-problema)

Uma escola técnica possui um sistema que registra alunos matriculados em diferentes cursos. A coordenação deseja analisar os dados para entender:

quais alunos participam de mais de um curso?
quais alunos estão em apenas um curso específico?
quais alunos não participam de determinadas turmas?

Para isso, você deverá utilizar consultas SQL avançadas para comparar resultados entre tabelas.

CREATE TABLE alunos (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  nome TEXT NOT NULL
);

CREATE TABLE curso_ds (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  aluno_id BIGINT REFERENCES alunos(id)
);

CREATE TABLE curso_adm (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  aluno_id BIGINT REFERENCES alunos(id)
);

INSERT INTO alunos (nome) VALUES
('Ana'),
('Bruno'),
('Carlos'),
('Daniela'),
('Eduardo'),
('Fernanda');

INSERT INTO curso_ds (aluno_id) VALUES
(1),
(2),
(3),
(5);

INSERT INTO curso_adm (aluno_id) VALUES
(2),
(3),
(4),
(6);

SELECT aluno_id FROM curso_ds
INTERSECT
SELECT aluno_id FROM curso_adm;

SELECT aluno_id FROM curso_ds
EXCEPT
SELECT aluno_id FROM curso_adm;

SELECT aluno_id FROM curso_adm
EXCEPT
SELECT aluno_id FROM curso_ds;
